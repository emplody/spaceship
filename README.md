- getPadNumber

  > geFormattedNumberString What you want

- normalizeKeyword

- objToParam

- paramsToObj

  > when we develop web service, we frequently get data params and change to object

- scriptLoader

  > When we want to dynamically import scripts.

- toTimeString

  > get TimeStamp from system time stamp

- uniqueKey
